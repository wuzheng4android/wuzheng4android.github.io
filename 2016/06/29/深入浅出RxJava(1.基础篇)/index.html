深入浅出RxJava（一：基础篇）

  • 基础
    • Observables(被观察者,事件源)
    • Subscribers(观察者)
    • Observables发出事件,Subscribers处理,一个Obervable可以发出多个事件,直到结束或者出错,每发出一个事件,就会调用Subriber的onNext()方法,最后调用Subscriber.onComplete()或onError()结束
  • Hello Word
    • 创建一个Obervable对象
      <pre><code>
      Observable myObservable = Observable.create(
      new Observable.OnSubscribe() {
      @Override
      public void call(Subscriber<? super String> sub) {
      //Observable对象发出一个HelloWord字串
sub.onNext("Hello, world!");
//结束
sub.onCompleted();
      }
      }
      );
    • 创建一个Subscriber来处理Observable对象发出的字符串 

      Subscriber mySubscriber = new Subscriber() {
      @Override
      public void onNext(String s) { System.out.println(s)
      ; }
      @Override
      public void onCompleted() {
      }
      @Override
      public void onError(Throwable e) { }
      };
